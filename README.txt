GIRLS FOR KNOWLEDGE WEBSITE

Files:
- index.html
- style.css

How to preview:
1. Keep index.html and style.css in the same folder.
2. Open index.html in a web browser.

Important replacements before publishing:
- Replace the GFK logo placeholder with the real logo.
- Replace WhatsApp, Instagram, Facebook, and email information.
- Replace YOUR_DONATION_LINK with a verified donation page.
- Replace sample text with the organization’s official mission and programs.
- Replace sample images with approved organization photographs.

Instagram images:
Use only photographs the organization owns or has permission to publish. Download the original image and place it inside an images folder, then change the image source in index.html, for example:
<img src="images/program-photo.jpg" alt="Girls participating in a learning program">
